# Gestiunea Stocurilor
Aplicația realizată automatizează și optimizează întregul proces de gestiune a stocurilor, permițând evidența, monitorizarea și controlul intrărilor, ieșirilor, transferurilor și documentelor asociate fiecărui produs.Gestionarea produselor (adăugare, editare, clasificare).

# Funcționalități
-Administrarea depozitelor și gestionarilor./n
-Repartizarea produselor în depozite și stabilirea stocului inițial.
-Generarea documentelor: NIR, aviz, bon transfer, bon consum, factură.
-Editarea și consultarea documentelor existente.
-Afișarea stocului curent pentru orice produs.
-Generarea balanței stocurilor.
-Istoricul complet al unui produs.
-Fișa de magazie.
-Calcul automat CMP pentru evaluarea ieșirilor.
